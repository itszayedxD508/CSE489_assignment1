VangtiChai - Money Change Calculator
=====================================

CSE 489: Mobile Application Development - Assignment 1

Description:
VangtiChai is a Flutter app that lets you enter a money amount via a
custom numeric keypad and calculates the change for that amount in
terms of Taka notes (500, 100, 50, 20, 10, 5, 2, 1).

Features:
- Custom numeric keypad (digits 0-9 and Clear button)
- Real-time change calculation displayed in a table
- Responsive layout for portrait and landscape orientations
- Scales gracefully between phone and tablet screen sizes
- State preservation on orientation change

Technology:
- Flutter (Dart)
- Minimum API Level: 21 (Android 5.0 Lollipop)

Tested Devices:
- Pixel XL (411x731dp) - Portrait and Landscape
- Nexus 10 (800x1280dp) - Portrait and Landscape
- Pixel 7 (412x732dp) - Portrait and Landscape
- Pixel 5 (393x851dp) - Portrait and Landscape
- Samsung Galaxy S21 (360x800dp) - Portrait and Landscape
- iPad Pro 11-inch (834x1194dp) - Portrait and Landscape

How to Run:
1. Ensure Flutter SDK is installed
2. Run 'flutter pub get' to install dependencies
3. Run 'flutter run' to launch on a connected device or emulator
